# Super App AI — Sidekick create action

Use this action when the merchant wants to build something new that Super App AI
generates: a storefront popup, banner, bundle, upsell, discount or cart-transform
Function, recommendation block, or an admin / POS / checkout / customer-account
block.

Pass the merchant's request as `prompt`, phrased the way they said it (for example
"a spin-to-win popup that matches my brand"). This opens the Super App AI generator
with that prompt pre-filled.

Nothing is created or published by opening the generator. The merchant reviews the
AI-generated options, customizes, and confirms inside Super App AI before any module
is saved or goes live. Do not describe this action as if it publishes a module.
